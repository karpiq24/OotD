# OotD Campaign Wiki

This is the digital home for the **Odyssey of the Dragonlords (OotD)** D&D campaign. 

It is a static site generated by [Quartz](https://quartz.jzhao.xyz/), styled as a wiki for easy navigation of our sessions, lore, and world atlas.

### Prerequisites
- Node.js v18+

### Running Locally
To preview the wiki on your machine:

```bash
npx quartz build --serve
```

Then open `http://localhost:8080` in your browser.

## ðŸ“‚ Project Structure

- **`content/`**: The heart of the wiki. All markdown files live here.
    - `01-Sessions`: Recaps of our adventures.
    - `02-People`: NPCs, PCs, and Factions.
    - `03-Locations`: Maps and geography.
    - `04-Items-and-Loot`: Inventory and artifacts.
    - `05-Lore`: History and religion.
    - `06-Rules`: House rules.
- **`quartz.config.ts`**: Configuration for the site (plugins, theme, etc.).
- **`quartz.layout.ts`**: Page layout definitions.
